# Speech-Based COPD Detection using Acoustic Analysis

## Project Description:

This project explores a non-invasive approach for detecting Chronic Obstructive Pulmonary Disease (COPD) using speech-based acoustic analysis. The system uses voice recordings to extract meaningful acoustic features and applies machine learning techniques to distinguish between COPD and healthy speech patterns.

The goal of this work is to support early screening and assist in healthcare analysis using signal processing and machine learning methods. This project is intended for academic and research purposes only.
