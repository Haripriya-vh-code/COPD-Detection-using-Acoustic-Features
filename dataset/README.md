# Speech-based COPD Detection Dataset

## Project Title
Speech-based Detection of Chronic Obstructive Pulmonary Disease using Acoustic Biomarkers

## Authors
Haripriya V H, Dakshanaa Shree Selvakumar, Seethalakshmi S, M. P. Actlin Jeeva  
Department of Electronics and Communication Engineering  
Sri Sivasubramaniya Nadar College of Engineering, Chennai, India  


## Dataset Description

The dataset contains speech recordings collected for the purpose of non-invasive detection of Chronic Obstructive Pulmonary Disease (COPD) using acoustic analysis.

---

## Data Collection Details

- Total participants: **68**
  - COPD patients: **35 (clinically diagnosed)**
  - Healthy controls: **33**
- Age group: **45–75 years**
- Language: **Native Tamil speakers (South India)**
- Speech tasks:
  - Sustained vowels: **/a/, /e/, /i/, /o/, /u/**
  - Spontaneous speech samples

---

## Recording Specifications

- Microphone: Sennheiser head-mounted microphone
- Sampling frequency: **44 kHz**
- Preprocessing:
  - Amplitude normalization
  - Resampling to **16 kHz**

---

## Dataset Statistics

- Total utterances: **455**
  - COPD group: **225**
  - Healthy group: **230**

---

## Features Extracted

The dataset is designed for acoustic biomarker analysis and includes:

### Spectral Features
- Formants (F1, F2, F3)
- Formant ratios 
- Spectral flux
- Subharmonic ratio (SHR)

### Temporal Features
- Jitter
- Shimmer
- Breathing energy
